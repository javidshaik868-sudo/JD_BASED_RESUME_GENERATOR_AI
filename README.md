# JD_BASED_RESUME_GENERATOR_AI

JD_BASED_RESUME_GENERATOR_AI is an AI-powered application built using Python and Streamlit that analyzes job descriptions, extracts required skills, and matches them with resumes. It identifies skill gaps and provides intelligent AI-based recommendations to improve resumes and increase job matching accuracy. The system helps both recruiters and job seekers make better hiring and career decisions through an interactive dashboard.

---

## 🚀 Features

- Job Description (JD) analysis
- Resume parsing and skill extraction
- Skill gap detection
- AI-based resume improvement suggestions
- Resume vs JD matching score
- Interactive dashboard for users
